# Frontend Intern Operations System

A vanilla JavaScript Single Page Application (SPA) for managing interns.

## Structure

- `index.html`: Main entry point and app shell.
- `css/`: Stylesheets.
  - `reset.css`: Browser normalization.
  - `layout.css`: Structural layout styles.
  - `components.css`: Reusable component styles.
- `js/`: JavaScript modules.
  - `state.js`: Centralized state management.
  - `fake-server.js`: Mock backend API.
  - `rules-engine.js`: Business logic rules.
  - `validators.js`: Form validation helpers.
  - `renderer.js`: DOM manipulation and rendering.
  - `app.js`: Application logic and router.

## Getting Started

1. Open `index.html` in your web browser.
2. Navigate using the links in the header.
